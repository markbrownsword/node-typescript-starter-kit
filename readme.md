## Node Typescript Starter Kit
### Get started with Node.js, Express, TypeScript and Docker

### Prerequisites
Install Node.js v8+ and Docker

### Setup
Clone the repo and install dependencies
```bash
npm install
```

### Development

```bash
npm run build:live
```

### Test

```bash
npm test
```

### Run

```bash
npm start
```

### Build Docker image

```bash
docker build .
```
